# ggWebGL: WebGL Rendering Backend for ggplot2

`ggWebGL` provides a browser-native WebGL path for a focused `ggplot2`
subset. The current package supports dense 2D point, line, trajectory,
and raster rendering, fixed-scale faceting, htmlwidget export, Shiny
output bindings, and optional static snapshot/export surfaces. Backend
integrations must hand over normalized primitive-scene payloads through
explicit adapter or bridge boundaries.
